## الهدف

صفحة تحويل + ثقة لولي الأمر (أساسًا) ثم الطالب: فهم سريع، اختيار دولة، حجز/حصة مجانية أو دخول — دون عرض كل ميزات المنصة دفعة واحدة.

---

## القرارات المعتمدة

| قرار | الاختيار |
|------|----------|
| CMS الهبوط | **مرحلة لاحقة** — المرحلة الحالية فرونت فقط |
| الإحصاءات | **نصوص ثقة** دائماً؛ الأرقام الحقيقية تظهر فقط عند نضج العتبة |
| الهيرو | **مزيج**: خلفية صف/طلاب + طبقة واجهة المنصة |

### عتبة الإحصاءات (`STATS_MIN`)

تظهر أرقام API فقط إذا تحقق أحد:

- طلاب ≥ **50**، أو
- معلمون ≥ **5**، أو
- دول ≥ **3**

المصدر: `GET /api/public/stats`

---

## هيكل الأقسام (من الأعلى)

1. **Nav** — شعار، اختيار دولة، روابط مرساة، تسجيل الدخول → `/login`
2. **Hero** — براند + عنوان + جملة + CTA (حجز / حصة مجانية) + خلفية + mock واجهة
3. **Trust** — 4 نصوص ثقة (+ إحصاءات إن نضجت)
4. **Audience** (`#audience`) — ولي أمر / طالب / معلم
5. **Product proof** (`#features`) — 3 محاور + صورة إثبات
6. **Banners** — من API حسب الدولة (إن وُجدت)
7. **Steps** (`#steps`) — 3 خطوات
8. **FAQ** (`#faqs`) — من API المنصة
9. **CTA band** — حصة مجانية
10. **Footer** — شعار + سوشيال حسب الدولة
11. **Lead modal** — `POST /api/leads`

---

## نصوص الثقة (الحالية)

1. متابعة يومية لولي الأمر: حضور، درجات، وتقارير واضحة.
2. حصص مباشرة تفاعلية مع تسجيل للمراجعة.
3. مساعد ذكي يوجّه الطالب بالتلميح لا بالإجابة الجاهزة.
4. نقاط ودوري وتحديات تحفّز الالتزام دون تشتيت.

---

## نصوص الهيرو (الحالية)

- **عنوان:** تعلّم بثقة. / تابع بوضوح.
- **دعم:** حصص مباشرة، متابعة لولي الأمر، ومساعد ذكي — في منصة عربية واحدة.

## APIs المستخدمة

| Method | Endpoint | ملاحظة |
|--------|----------|--------|
| GET | `/api/public/countries` | اختيار الدولة |
| GET | `/api/public/faqs` | أسئلة المنصة (`country_id` null) |
| GET | `/api/public/stats` | أرقام حقيقية |
| GET | `/api/public/banners?country_id=` | حسب الدولة |
| GET | `/api/public/social?country_id=` | حسب الدولة |
| POST | `/api/leads` | `source`: `book_now` \| `free_class` |

## مقاسات الصور المطلوبة

ضع الملفات في:  
`codes/frontend/public/landing/`  
(بعد الـ build تُنسخ تلقائياً إلى `dist/` ثم إلى `backend/public/landing/`)

| الملف | المقاس | الاستخدام |
|--------|---------|-----------|
| `hero-bg.jpg` | **1920×1080** | خلفية الهيرو (ديسكتوب) |
| `hero-bg-mobile.jpg` | **1080×1920** | خلفية الهيرو (موبايل) |
| `hero-ui.png` | **1200×750** | طبقة واجهة المنصة فوق الخلفية |
| `path-parent.jpg` | **800×600** | مسار ولي الأمر |
| `path-student.jpg` | **800×600** | مسار الطالب |
| `path-teacher.jpg` | **800×600** | مسار المعلم |
| `product-proof.jpg` | **1440×900** | إثبات المنتج |

**ملاحظات تصميم:**

- الهيرو: منطقة آمنة للنص يمين/وسط؛ لا تفاصيل مهمة في الأطراف.
- الموبايل: الوجه/الصف في الثلث العلوي.
- يفضّل WebP بنفس الأسماء + JPG/PNG احتياطي.
- وزن الهيرو المستهدف &lt; ~300KB بعد الضغط.
- بانر ترويجي من الأدمن (منفصل): يُفضّل **1440×360**.

حتى رفع الصور تظهر **placeholders** بالمقاس ومسار الملف في الواجهة.
